# Eval test cases

5 сценариев для ручной/агентной проверки скилла. Ожидаемые свойства —
`expected-properties.md`.

## Сценарий 1 — Гипотеза Римана

**Prompt:** «Предложить 100 направлений для доказательства гипотезы Римана.»

**Режим:** frontier (высокая ставка, нужен adversarial audit)

## Сценарий 2 — Теория трёх поколений из геометрии S³ × S⁶

**Prompt:** «Развить теорию трёх поколений из геометрии S3 × S6.»

**Режим:** frontier

## Сценарий 3 — Надёжность автономных ИИ-агентов

**Prompt:** «Повысить надёжность автономных ИИ-агентов.»

**Режим:** deep

## Сценарий 4 — Аномалии в геопространственной телеметрии

**Prompt:** «Найти способы обнаружения аномалий в геопространственной телеметрии.»

**Режим:** deep

## Сценарий 5 — Конверсия цифрового продукта

**Prompt:** «Найти 100 способов повысить конверсию цифрового продукта.»

**Режим:** deep (нет причин для frontier — это не фундаментальная научная
проблема, ожидание — НЕ вызывать научную театральность там, где достаточно
A/B-теста)
