# Domain lenses & generation operations

Читается на Шаге 4-5 SKILL.md. Не включать линзу только ради счёта — для
каждой выбранной объяснять её связь именно с текущей задачей.

**Минимум 12 из этих 25** должны быть выбраны и явно связаны с задачей на
Шаге 4. Выбор — не механический (не "первые 12 по списку"), а по реальной
релевантности к `target_system`/`primary_goal` из GoalContext.

- геометризация

Внутренний пул (Шаг 5) — рабочий материал, не показывается пользователю

выжившие, содержательно различные варианты.
