# Каталог кросс-доменных мостов

Стартовый набор проверенных мостов. Формат: **Куда** ← *Откуда* · почему (формальный изоморфизм) · проверка · где ломается.
Используй как затравку на Шаге 2 — но настоящая сила в НОВЫХ мостах под конкретную задачу.

> Дисциплина: каждый мост — это **структурный изоморфизм**, а не метафора. Прежде чем переносить, назови общую формальную структуру (отношения/ограничения/динамику).

## Системные (бонус)

- **Сложность кодовой базы ← геология (стратиграфия):** техдолг = слои отложений; `git blame` как геологический разрез. Слом: код переписываем, геологию — нет.
- **Управление знаниями ← экология (биоразнообразие):** silo = вымирание вида, transfer = миграция. Слом: «вымершие» знания восстановимы.
- **Распределённый консенсус ← теория аукционов:** Byzantine agreement ≡ аукцион с нечестными; mechanism design. Слом: высокая сложность реализации.

## Как пользоваться каталогом
1. На Шаге 2 пробеги по таблицам — есть ли структура, похожая на твою задачу?
2. Если да — это **затравка**, не ответ: проверь формальный изоморфизм под СВОЮ задачу.
3. Если нет — генерируй новые мосты с нуля (каталог не ограничивает).
4. Любой мост перед использованием → колонка «Где ломается» + Confidence scoring + `skeptic`.
