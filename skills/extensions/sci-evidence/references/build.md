# Sci-Evidence — Режим BUILD: "5 Путей к Доказательству"

## Принцип Консилиенса (Consilience)

**Консилиенс** = независимые линии доказательства сходятся к одному выводу.

Одна сильная корреляция — не доказательство.
Одно механистическое объяснение — не доказательство.
Один эксперимент — не доказательство.

**Три независимые линии** средней силы > **одна линия** высокой силы.

Почему: каждая линия имеет свои слабости. Если они независимы (разные данные, методы, логика) — их слабости не перекрываются. Это то, что делает доказательство устойчивым.

---

## Как использовать

Для каждого из 5 путей:
1. Оцени текущий статус: **Нет / Слабый / Умеренный / Сильный**
2. Укажи конкретное доказательство (если есть) с источником
3. Укажи следующий шаг: что конкретно добавит этому пути
4. Оцени независимость от других путей (важно для консилиенса)

## 🔬 Путь 1: Механистический — "Почему это должно работать"

**Вопрос:** Есть ли теоретическое обоснование почему X должно влиять на Y?

**Что это даёт:** Без механизма — это наблюдение. С механизмом — это объяснение.

**Типы механистических доказательств:**

| Тип | Пример | Сила |
|-----|--------|------|
| Физический/химический механизм | "Молекула А блокирует рецептор Б" | Сильный |
| Математическая модель с предсказаниями | "При параметрах X и Y уравнение предсказывает Z" | Сильный |
| Компьютерная симуляция | "Агентная модель воспроизводит эффект" | Умеренный |
| Аналогия с известным механизмом | "Аналогично как работает P в системе Q" | Слабый |
| Умозрительное объяснение | "Вероятно, потому что..." | Нет |

**Красные флаги:**
- Механизм придуман ПОСЛЕ наблюдения эффекта (ad hoc explanation)
- Механизм не генерирует новых предсказаний (не falsifiable)
- Нет промежуточных звеньев (mechanism is a black box)

**Зелёные флаги:**
- Механизм предсказал что-то до наблюдения
- Есть экспериментальное подтверждение промежуточных звеньев
- Механизм объясняет не только основной эффект но и паттерны исключений

## 📊 Путь 2: Наблюдательный — "Что говорят данные в природе"

**Вопрос:** Встречается ли в реальных данных паттерн, который предсказывает гипотеза?

**Что это даёт:** Экологическая валидность — эффект существует в реальных условиях.

**Типы наблюдательных доказательств:**

| Тип | Пример | Сила |
|-----|--------|------|
| Проспективная когорта | "3000 человек 10 лет, X предсказывает Y" | Сильный |
| Ретроспективный анализ | "Исторические данные: X коррелирует с Y" | Умеренный |
| Cross-sectional | "Сейчас: места с X имеют больше Y" | Умеренный |
| Case series / Case reports | "5 пациентов с X у всех был Y" | Слабый |
| Anecdotal / single case | "Я видел что X связан с Y" | Нет |

**Красные флаги:**
- Корреляция без контроля конфаундеров (→ Мир 2 из FALSIFY)
- Только cross-sectional (не понятна направленность во времени)
- Данные cherry-picked (выбраны только подтверждающие)

**Зелёные флаги:**
- Dose-response relationship (больше X → больше Y)
- Эффект устойчив в разных популяциях и периодах
- Паттерн соответствует форме предсказанной механизмом

## 🧪 Путь 3: Экспериментальный — "Что происходит при вмешательстве"

**Вопрос:** Если вмешаться (изменить X), изменится ли Y так, как предсказывает гипотеза?

**Что это даёт:** Причинность — не просто корреляция, а направленный эффект.

**Типы экспериментальных доказательств (по силе):**

| Тип | Контроль | Сила |
|-----|----------|------|
| RCT (double-blind, pre-registered) | Рандомизация + ослепление | Очень сильный |
| RCT (open label) | Только рандомизация | Сильный |
| Quasi-experiment (natural experiment, RD) | Частичная | Умеренный |
| A/B тест (производственный) | Рандомизация по пользователям | Сильный (в домене) |
| Ablation study (ML/AI) | Убираем компонент | Умеренный |
| Before-after (without control group) | Нет контрольной группы | Слабый |

**Красные флаги:**
- Отсутствие контрольной группы
- Нет pre-registration (анализ придуман после данных)
- Исход выбран post-hoc (multiple endpoints)
- Contamination / crossover (группы смешались)

**Зелёные флаги:**
- Pre-registered protocol соответствует анализу
- Эффект воспроизводится в нескольких независимых экспериментах
- Размер эффекта соответствует ожидаемому из механизма

## 🔁 Путь 4: Сравнительный — "Похожие случаи показывают то же"

**Вопрос:** В аналогичных системах, доменах, или периодах — наблюдается тот же паттерн?

**Что это даёт:** Обобщаемость и независимость от специфики конкретного контекста.

**Типы сравнительных доказательств:**

| Тип | Пример | Сила |
|-----|--------|------|
| Cross-domain replication | "То же в биологии что и в экономике" | Сильный |
| Animal model / model organism | "В мышах X вызывает Y" | Умеренный |
| Historical analogy | "В 1918 пандемии паттерн тот же" | Умеренный |
| Geographical comparison | "Страны с X имеют меньше Y" | Слабый |
| Developer/domain expert consensus | "5 экспертов независимо пришли к тому же" | Слабый |

**Красные флаги:**
- Аналогия поверхностная (похожие слова, разная механика)
- Сравниваемые случаи не независимы (одни и те же данные разными путями)
- Cherry-picked примеры (игнорируются контр-примеры)

**Зелёные флаги:**
- Одинаковый паттерн в независимо изученных системах
- Объяснение ПОЧЕМУ аналогия держится (не просто "похоже")
- Включает контр-примеры с объяснением почему там не работает

## 🎯 Путь 5: Предсказательный — "Гипотеза предсказала что-то новое"

**Вопрос:** Делала ли гипотеза нетривиальные предсказания, которые потом подтвердились?

**Что это даёт:** Самый сильный вид доказательства — гипотеза прошла "blind test".

**Почему это важно:** Подгонка данных (post-hoc fit) даёт нулевую информацию. Предсказание и подтверждение — максимальная информация.

**Типы предсказательных доказательств:**

| Тип | Пример | Сила |
|-----|--------|------|
| Нетривиальное предсказание подтверждено | "Из теории следовало X, нашли X" | Очень сильный |
| Out-of-sample prediction (held-out data) | "Модель предсказала Y на новых данных точнее baseline" | Сильный |
| Pre-registered prediction | "До анализа написали что ожидаем X" | Сильный |
| Qualitative prediction | "Ожидали что при A будет B, и было" | Умеренный |
| Ретроспективное "предсказание" | "Конечно, теория объясняет X (после того как X известен)" | Нет |

**Красные флаги:**
- Все предсказания делаются post-hoc (под известный результат)
- Предсказания тривиальны (предсказывают то, что любая теория предскажет)
- "Предсказание" содержит free parameters которые подогнали

**Зелёные флаги:**
- Специфическое количественное предсказание подтверждено (не просто "X увеличится")
- Предсказание было неочевидным на момент его формулировки
- Несколько независимых предсказаний подтверждены

## Подсчёт Consilience Score

**Scoring:**
- Нет = 0
- Слабый = 0.5
- Умеренный = 1.5
- Сильный = 2.5

**Максимум:** 5 × 2.5 = 12.5 → нормализуем к 10 (делим на 1.25)

**Поправка на независимость:** Если два пути используют одни и те же данные → зачесть как 1.5× вместо 2×

**Интерпретация:**

| Score | Интерпретация | Действие |
|-------|--------------|----------|
| 0–2 | Гипотеза голая — нет доказательной базы | Начни с Пути 1 (механизм) |
| 2–4 | Начальные признаки — недостаточно для утверждений | Добавь 2–3 независимых пути |
| 4–6 | Умеренная поддержка — можно публично обсуждать | Найди самый слабый путь, усиль |
| 6–8 | Сильная позиция — готово к peer review | Сфокусируйся на Пути 5 (предсказания) |
| 8–10 | Очень сильная — высокий уровень уверенности | Готово к public claim |

## Приоритет Путей (если ресурсы ограничены)

**Если можно сделать только один путь:**
→ Путь 3 (Экспериментальный) — дает наибольшую информацию о причинности

**Если уже есть механизм (Путь 1):**
→ Следующий — Путь 5 (Предсказательный) — проверь нетривиальное предсказание

**Если нет возможности для эксперимента:**
→ Пути 1+2+4 (механизм + наблюдение + сравнение) = умеренная позиция

**Один путь никогда не достаточен для сильного утверждения.** Минимум 2 независимых пути для публичных claims.
