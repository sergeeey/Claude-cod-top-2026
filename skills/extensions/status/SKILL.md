# Session Status

Собери дайджест в 3 блока. Читай источники последовательно:

## Шаг 1 — Читай источники

1. **Git лог** — `git log --oneline -10` в текущем репо (что реально закоммичено)
2. **activeContext.md** — `.claude/memory/activeContext.md` в текущем проекте (если есть)
3. **Открытые планы** — `~/.claude/plans/*.md` (если есть — заголовок + статус)
4. **Текущая ветка** — `git branch --show-current` (контекст работы)

Если файл не найден — пропусти, не сообщай об ошибке.

## Шаг 2 — Формат вывода

Выводи ТОЛЬКО этот блок, без лишних слов до и после:

```
📍 Ветка: [название ветки] | [дата]

✅ СДЕЛАНО (эта сессия / последние коммиты)
• [факт из git лога или activeContext]
• ...

🔄 В ПРОЦЕССЕ
• [незакрытые задачи из activeContext / открытые планы]
• ...

⏳ ОСТАЛОСЬ / ВИСИТ
• [явные TODO, незакрытые пункты планов, упомянутые но не сделанные вещи]
• ...

🔴 ГОРИТ (если есть)
• [CI fail, дедлайны, критические баги]
```

## Правила

- Максимум 5 пунктов в каждом блоке
- Если блок пустой — не выводи его совсем
- Пункты — одна строка, конкретно (не "работали над кодом", а "починили баги C1-C3 в hypothesis-lab")
- 🔴 ГОРИТ — только реально критичное (CI fail, дедлайн сегодня, прод сломан)
- После блока — одна строка: **Следующий шаг:** [самое важное сейчас]
