# SNR Daily Triage — Пятница 16:00

**Главный outcome:** разблокировать CI к вечеру (мёрж сегодня).  
**Метод:** SNR = Signal / Noise по формуле из скила `snr` v2.0.

---

## Формула и шкала

```
SNR = Signal Value / Noise Cost

Signal = Impact × Compounding × Leverage × Reusability × Confidence
Noise  = Time + Fragmentation + Coordination + Rework + Governance
```

| SNR    | Решение   |
|--------|-----------|
| > 8    | KEEP      |
| 4–8    | SIMPLIFY  |
| 2–4    | BATCH     |
| < 2    | ELIMINATE |
