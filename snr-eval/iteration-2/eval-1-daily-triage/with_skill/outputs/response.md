**Задача 3 — CHANGELOG.md (SNR 3.3 → BATCH):**
Impact=2: CHANGELOG важен для истории, но не влияет на outcome сегодня. Reuse=2: станет шаблоном паттерна. Confidence=5: задача полностью определена. Time=2: 30–40 мин. **Задача 4 — Review PR (SNR 4.4 → SIMPLIFY):**
Impact=3: code quality имеет значение, но явно сказано "не блокирует никого". Fragmentation=3: 300 строк = глубокое погружение, разорвёт контекст. Rework=2: после review может быть обратная связь. **Задача 6 — Ретроспектива спринта (SNR 21.6 → KEEP, но не сегодня):**
